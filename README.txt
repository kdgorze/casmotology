IMAGES FOLDER — Casmotology
===========================

Drop your photos in here and reference them in index.html.

Recommended naming convention:
  hero-1.jpg       → top-left hero grid photo
  hero-2.jpg       → top-right hero grid photo
  hero-3.jpg       → bottom-left hero grid photo
  hero-4.jpg       → bottom-right hero grid photo

  work-1.jpg       → Portfolio card 1 (Balayage)
  work-2.jpg       → Portfolio card 2 (Precision Cut)
  work-3.jpg       → Portfolio card 3 (Updo)
  work-4.jpg       → Portfolio card 4 (Full Color)
  work-5.jpg       → Portfolio card 5 (Highlights)
  work-6.jpg       → Portfolio card 6 (Blowout)

  about.jpg        → Your portrait photo in the About section

How to use them in index.html:
  Replace:  <div class="work-card-img">Add photo</div>
  With:     <div class="work-card-img"><img src="images/work-1.jpg" alt="Balayage Color" /></div>

Tips:
  - Keep images under 500KB for fast loading (use squoosh.app to compress)
  - Use .jpg for photos, .png for images with transparency
  - Recommended dimensions: 800×1000px for portrait/work shots
